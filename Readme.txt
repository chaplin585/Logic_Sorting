Logic_Sorting

Usage:
Put in the root of the C drive

About the program :
1)Run the main file.py in the src folder
2) To update the data, click the Update data button
3) The latest updates are indicated on the left on the information board
4) To view the sorted data, press the Main Tabs key

For developers:
To get full information from unit tests, you need to write the python -m unit test command in the console (open in the test folder)
test_parsing.py (to test the parser) python -m unit test test_sorting.py (to check the sorting operation)
